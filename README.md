# Анализ социально-экономических показателей России

Проект разработан в рамках лабораторной работы №3 по дисциплине «Технологии программирования» на тему «Системы контроля версий (VCS)». Приложение предоставляет графический интерфейс для загрузки и анализа статистических данных по двум направлениям: доля автомобильных дорог, не отвечающих нормативным требованиям, по субъектам РФ и уровень инфляции в России. Программа строит графики, выполняет прогнозирование методом скользящей средней и позволяет экспортировать результаты.

## Функциональные возможности

### Вариант 16. Доля плохих дорог

- Загрузка данных из CSV-файла с информацией о проценте дорог, не отвечающих нормативам, по десяти субъектам Российской Федерации за 2010–2026 годы.
- Отображение всех записей в табличном виде с возможностью сортировки по субъекту и году.
- Построение линейных графиков динамики доли плохих дорог для каждого субъекта.
- Интерактивное масштабирование графиков (выделение области) и кнопка сброса масштаба.
- Аналитический расчёт: определение субъектов с максимальным и минимальным снижением доли плохих дорог за наблюдаемый период.
- Прогнозирование методом экстраполяции по скользящей средней: пользователь задаёт период сглаживания, выбирает субъект, после чего программа вычисляет прогноз на соответствующее количество лет и отображает его на графике пунктирной линией другого цвета.
- Экспорт графика в графический файл формата PNG.

### Вариант 10. Инфляция

- Загрузка данных об уровне инфляции в России по годам (2010–2024) из CSV-файла.
- Табличное представление и график инфляции.
- Прогнозирование уровня инфляции методом скользящей средней на N лет вперёд.
- Расчёт возможной стоимости товара через N лет на основе введённой текущей цены и прогнозируемой инфляции.
- Экспорт графика в PNG и интерактивное масштабирование.

## Структура репозитория и веток

Репозиторий организован в соответствии с моделью Git Flow:

- **main** — стабильная версия, готовая к демонстрации.
- **dev** — основная ветка разработки, в которую сливаются все готовые фичи.
- **feature/roads** — ветка для реализации варианта 16 (доля плохих дорог). Автор: Мирза А.Р.
- **inflation** — ветка для реализации варианта 10 (инфляция). Автор: Ионкина В.Д.

Каждый участник вёл работу в своей функциональной ветке, регулярно делал коммиты и отправлял изменения на GitHub. Слияние кода производилось через Pull Request с обязательным код-ревью.


## Используемые технологии

- Язык программирования: C#

## Как запустить проект

1. Склонируйте репозиторий:https://github.com/seee902u/VCS_Lab3
2. Откройте файл решения `EconomicIndicatorsApp.sln` в Visual Studio.
3. Убедитесь, что целевая платформа .NET Framework (4.7.2 или 4.8) установлена.
4. Соберите решение (Ctrl+Shift+B) и запустите (F5).
5. Для загрузки данных используйте файлы из папки `Data` внутри каталога программы.
